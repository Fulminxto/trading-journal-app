# DESIGN REVIEW

Before approving any screen, evaluate:

---

## Structural Review

- Is hierarchy clear?
- Is there one dominant truth?
- Is spacing coherent?
- Is visual weight balanced?
- Is information breathable?

---

## Brand Review

- Does it feel like the ecosystem?
- Does it respect product DNA?
- Does it match Voltis/Vitalis personality?
- Does it feel handcrafted?
- Does it feel premium?

---

## Experience Review

- Does it feel alive?
- Is motion useful?
- Is the reveal staged correctly?
- Does interaction feel smooth?
- Does it guide the eye?

---

## Noise Review

Remove:

- unnecessary cards
- duplicated data
- weak labels
- visual clutter
- secondary noise

Always simplify before adding.

---

## Premium Review

Ask:

Would a premium user trust this?

Would this feel expensive even if free?

Would this make someone say:
"this is different."

If not:
rework.
